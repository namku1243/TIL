## 조장

* 단톡방
* 프로젝트 설정
  * 로컬 폴더
    * 프로젝트 설정 (web 기본코드, ... )
    * 로컬 폴더 => git 저장소
  * 원격저장소 (github)
    * `end` 원격 저장소 만들고 push!
    * 조원 초대하기

## 조원

* 원격저장소 초대 받고

* clone

  ```bash
  $ git clone https://github.com/mongchigu/end.git
  # 받아오면, 원격 저장소 이름의 폴더가 생겨있어요
  $ cd end0205
  (master) $
  ```

  * 코드작성하고(끝말잇기)
  * commit
  * push
  * 조장에게 push했다고 알리기 (zoom 채팅에 "저 push했어요~")
