# Branch 명령어

* 브랜치 생성

  ```bash
  $ git branch 브랜치명
  ```

* 브랜치 목록

  ```bash
  $ git branch
  ```

* 브랜치 이동

  ```bash
  $ git checkout 브랜치명
  (브랜치명) $
  ```

* 브랜치 생성 및 이동

  ```bash
  $ git checkout -b 브랜치명
  ```

* 브랜치 병합

  ```bash
  (master) $ git merge 브랜치명
  ```

  * `master` 브랜치에 `브랜치명` 을 병합

* 브랜치 삭제

  ```bash
  $ git branch -d 브랜치명
  ```
